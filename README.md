# C# Programs Collection
This repository contains a collection of C# programs and exercises.
Each file demonstrates fundamental concepts of programming in C#, including loops, conditionals, recursion, arrays, and more.

## 📌 Contents
- test input number is odd or even
- Input number is positive, negative, or zero
- find largest among three number using nested if statement
- read (0-9) and display in respective word using switch case statement
- check input number is prime or not
- check whether given number is Armstrong or not
- check input number is perfect or not
- find nth term of Fibonacci series
- find sum of integer until user input negative number
- compute cube of first n natural number using goto statement
